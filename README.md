# Theme Toggle Extension

A simple Visual Studio Code extension that adds a button to the status bar for quickly switching between themes.

## Features

- Toggle between themes with one click  
- Shows a short status bar message after switching   

 ## How to Use 
 
1. Install the extension by using the included `.vsix` file: 
    - Open the Command Palette in VS Code 
    - Run **Extensions: Install from VSIX...** 
    - Select the provided `.vsix` file 
2. Reload VS Code if prompted. 
3. Click the theme button in the status bar to switch themes. 

## Development To run the extension in a development window: 

1. Open the project folder in VS Code. 
2. Press **F5** to launch the Extension Development Host. 
3. Test the extension in the new window.
